PAT甲级1131——1135

1131[DFS]

Subway Map

题目输入一个地铁图（就是无向图），然后对结点进行编号。之后输入站点进行查询，输出最短路线（经过的站点最少），如果经过站点一样输出换乘次数最少的。

思路：由于是稀疏图，故用邻接表存储，即存储每个顶点的邻接顶点。然后针对每一次查询用一次DFS搜索即可。另外，用二维数组line打表记录两个站点之间是几号线，根据这个好统计换乘次数。

1132[字符串分割]

Cut Integer

思路：一目了然。

1133[链表分段排序]

Splitting A Linked List

思路：用点小技巧，分多次for循环即可！

1134[图论概念]

Vertex Cover

思路：判断是不是顶点覆盖，根据定义判断即可。用邻接表比较好！

1135[红黑树]

Is It A Red-Black Tree

思路：红黑树判断，考试的时候没做出来 ，主要没理解透。其实题目中说的第三条是废话，它的意思是指空结点都是黑的，但是这个题而言没啥用的，但还是成功让我发呆了2小时没做出来。咳咳，根据先序序列建立树，然后DFS之判断“双红”和黑结点个数问题。
