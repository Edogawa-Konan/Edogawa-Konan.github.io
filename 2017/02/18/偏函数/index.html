偏函数

2017-02-18
  • 编程

Python的functools模块提供了很多有用的功能，其中一个就是偏函数（Partial function）。说白点很像c++的函数适配器

当函数的参数个数太多，需要简化时，使用functools.partial可以创建一个新的函数，这个新函数可以固定住原函数的部分参数，从而在调用时更简单。

例如：

import functools
int2=functools.partial(int,base=2)

print(int2('101011'))
print(int2('54',base=10))

注意partial填充的值是可以被覆盖的，如上面最后条语句。

最后，创建偏函数时，实际上可以接收函数对象、args和**kw这3个参数。
例如，当传入max2 = functools.partial(max, 10),实际上会把10作为args的一部分自动加到左边，亦即当调用max2(5,6,7)时会变成args=(10,5,6,7),max2(*args)
