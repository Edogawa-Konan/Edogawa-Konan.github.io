Java基本语法

2017-05-30
  • 编程

java基本语法概述，类比c++。

前言：java是纯面向对象的编程语言，这意味着所有的函数都是属于某个类的方法（c++称作成员函数），因此java中的 main方法必须有一个外壳类且main方法必须是public和static的。

Java基本语法and错误笔记

注释

java有三种注释形式，前两种/* */和//完全类似c++自不必再言，第三种是/** */会生成文档，应该和python的文档差不多。

数据类型

整型

java中整型的范围与运行java代码的机器无关（这一点比c/c++强喽），整型有：

int short long byte
4 2 8 1

长整型后缀L，十六进制前缀0x，八进制前缀0，二进制前缀0b。数字字面量可以加下划线如1_000_000只是为了方便读，编译器自动忽略它们。java没有任何无符号类型。

浮点类型：

float double
4 8

float类型后缀F，没有默认是double，也可以加后缀D表示double。常量Double.POSITIVE_INFINITY、Double.NEGATIVE_INFINITY、Double.NaN分别表示正无穷大、负无穷大和不是一个数字。可以用Double.isNaN()判断是不是非数值，不能用==因为所有非数值的值都认为是不相同的。

布尔boolean类型：

和c++一样两个值true和false，但是整型值和布尔值之间不能进行相互转换。c++中数值或指针可以代替boolean值。

常量

用关键字final标识一个数值变量，表示这个变量只能被赋值一次，java中final如果修饰引用变量，代表这个引用变量的指向不能变，但是内容可以变！

建议常量名全大写。

运算符

这部分和c++很像，&与、|或、^异或、~非，>>右移和<<左移，>>>用0填充高位，>>用符号位填充高位。这些位运算符都工作在位模式下，操作的是补码。移位只会在1到32或64之间，多于这个区间的会先求余，取决于操作数是long还是int。左移是乘，右移是除。

数学计算

Math类有很多方法，可以用于计算。数值类型之间会发生自动转换，当然也可以强制类型转换：比如（int）x。如果想对浮点数求其最接近的整数，可以用Math.round(x)，返回的结果是long类型。

字符串

java中没有字符串类型，只有一个预定义类String。这个类有很多成员函数可以完成各种操作，比如substring求子串。String类重载了+运算符，java本身不支持运算符重载，这也是唯一的一个。String本质上算是一个不可变对象，类比python的字符串。==不能用于判断字符串相等，意为判断两个字符串是否在同一位置上，应该用equals（）。此外，compareTo方法与c的strcmp一样。

注意，空串“”是长度为0的字符串，str.length==0且str.equals("")

如果需要由较短的字符串构建字符串，那么每次连接字符串耗时又浪费空间。可以用StringBuild类。

输入输出

标准输入输出

通过控制台输入，需要构造一个Scanner对象，并且与标准输入流System.in进行关联。Scanner类定义在java.util包中。

文件输入输出

要对文件进行读取，可以用一个File对象构造一个Scanner对象。
Scanner in=new Scanner(new File("e:\\hehe.txt"));
写文件需要构造一个PrintWriter对象，如下：
PrintWriter out=new PrintWriter("hehe.txt");

块作用域

java不能在嵌套的两个块中声明同名的变量，虽然在c++中会内部屏蔽外部。

for（variable:collection） statement
定义一个变量用于暂存集合中的每一个元素并执行相应的语句，collection必须是数组或实现了Iterable接口的类对象。这种形式可以让你不必操心边界问题，c++11也有这种。

数组

创建一个数组需要和创建对象一样用new操作符，如果不进行初始化，数字数组所有元素都初始化为0，boolean数组的元素会初始化为false，对象数组的元素则初始化为null。

int[] a=new int[10]

java创建数组变量时不能只能大小，即int a[10]这种是错误的！
